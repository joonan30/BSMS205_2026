Gene Regulation

Chapter 29 · Methods and Applications
Welcome to Chapter twenty-nine, Gene Regulation, Methods and Applications. Last chapter we learned the principles — promoters, enhancers, three-dimensional chromatin loops. Today we look at the experimental methods that actually let us measure all of this in real cells. We will cover R N A-seq, single-cell R N A-seq, ChIP-seq, ATAC-seq, and CUT and RUN. By the end you will know which tool to use for which question.
Today's central question

How do we measure
gene regulation?

We finished the previous chapter knowing that gene regulation has many layers — transcription factor binding, chromatin accessibility, histone marks, three-dimensional architecture. Today we ask the practical question. How do we measure these things? What is the experimental toolkit? Each method gives a different layer of regulatory information. Used together, they reveal the complete regulatory logic of any cell type.

§ 1

RNA-seq

Let's start with R N A-seq.

RNA-seq workflow

  1. Extract total RNA from cells or tissue
  2. Reverse transcribe → cDNA · fragment
  3. Add adapters · sequence (50–150 bp reads)
  4. Align to genome · count reads per gene
  5. Normalise · differential expression analysis
R N A-seq has five basic steps. One. Extract total R N A from your sample — tissue, cultured cells, or sorted populations. Two. Reverse transcribe the R N A into complementary D N A, then fragment it into pieces of about two hundred to five hundred base pairs. Three. Add sequencing adapters and run on an Illumina sequencer, generating millions of fifty to one hundred fifty base-pair reads. Four. Align the reads back to the reference genome and count how many reads mapped to each gene. Five. Normalise the counts and run statistical tests to identify differentially expressed genes between conditions. The result tells you which genes are on, which are off, and how their expression differs between samples.

Bulk RNA-seq · the big picture

This is the bulk R N A-seq workflow. R N A is extracted from a population of cells — could be a tissue biopsy, a culture dish, or a sorted population. Everything gets pooled. The output tells you the average gene expression profile across all the cells in the sample. That is powerful for comparing tissues, like liver versus brain. Or comparing conditions — healthy versus diseased. But bulk R N A-seq has a hidden cost. It averages over the cellular composition. If your sample is a mixture of neurons, astrocytes, and microglia, the result is a blend that does not reflect any one cell type.

Normalisation · why we need it

MethodWhat it corrects
CPMSequencing depth
FPKM / RPKMDepth + gene length
TPMDepth + length (better cross-sample)
DESeq2 normalisationDepth + RNA composition for DE testing
Raw read counts are not directly comparable across genes or samples. Two reasons. Longer genes generate more reads simply because there is more transcript to sequence. And different samples may have been sequenced to different depths. So we normalise. C P M, counts per million, scales by total sequencing depth. F P K M and R P K M correct for both depth and gene length. T P M is similar but rescaled so all values in a sample sum to one million, which makes cross-sample comparisons cleaner. For differential expression analysis specifically, tools like DESeq2 use a more sophisticated median-of-ratios method that also corrects for R N A composition.
§ 2

Single-Cell RNA-seq

Bulk R N A-seq averages across cells. Single-cell R N A-seq does not. Let's see how.

The droplet trick

  • Encapsulate each cell in a tiny oil droplet
  • Each droplet has a gel bead with a unique barcode
  • Cell lyses in droplet · mRNA tagged with cell barcode during RT
  • Pool · sequence · de-multiplex by barcode
Single-cell R N A-seq uses a beautiful microfluidic trick. Each cell is encapsulated in a tiny oil droplet along with a gel bead carrying a unique D N A barcode. When the cell lyses inside the droplet, its m R N A is captured and reverse-transcribed, with the cell's barcode attached to every c D N A molecule. After all cells are processed, you break the droplets, pool the c D N A, and sequence everything. Each read carries the cell barcode, so you can trace it back to its cell of origin. The most common platform is ten-X Genomics, which can process about ten thousand cells per run.

Bulk vs single-cell

This figure shows the single-cell workflow side by side with bulk. Bulk gives you one average profile per sample. Single-cell gives you one profile per cell — typically thousands or tens of thousands of cells per experiment. The output is a cell-by-gene matrix. Each row is a cell. Each column is a gene. Each value is the expression of that gene in that cell. From this matrix you can do dimensionality reduction with U MAP or t-S N E, cluster cells by similarity, and identify cell types. In brain tissue, single-cell R N A-seq can resolve dozens of distinct neuronal subtypes that were completely invisible in bulk experiments.

Single-cell R N A-seq reveals cell identity through marker gene expression. T cells in blood are identified by C D three, C D four, and C D eight. B cells by C D nineteen and M S four A one. In the brain, excitatory neurons express S L C seventeen A seven, the gene for the vesicular glutamate transporter. Inhibitory neurons express G A D one, glutamate decarboxylase. Astrocytes express G F A P. Microglia express C X three C R one. All of these cells share the same genome. What makes them different is their gene expression program. That is the take-home message — cell identity is defined by which genes are read, not by which D N A is present.
§ 3

The Histone Code

Now let's shift from R N A to chromatin. The protein machinery that controls D N A access. We start with histone modifications.

The chromatin code

MarkMeaning
H3K4me3Active promoter
H3K27acActive enhancer or promoter
H3K4me1Enhancer (active or poised)
H3K36me3Active gene body · elongation
H3K27me3Polycomb repression
H3K9me3Constitutive heterochromatin
Histones carry chemical modifications that work as a combinatorial code. Six marks dominate the literature. H three K four me three — trimethylation of histone three lysine four — marks active promoters. H three K twenty-seven ac marks active enhancers and promoters. H three K four me one marks all enhancers, both active and poised. H three K thirty-six me three is found in gene bodies during active transcription. H three K twenty-seven me three is the repressive mark deposited by Polycomb complexes. And H three K nine me three marks constitutive heterochromatin — permanently silenced regions like centromeres. Different combinations of these marks define different chromatin states.

Combining marks · chromatin states

  • Active promoter · H3K4me3 + H3K27ac
  • Active enhancer · H3K4me1 + H3K27ac
  • Poised enhancer · H3K4me1 (no H3K27ac)
  • Polycomb-repressed · H3K27me3
  • Heterochromatin · H3K9me3
Chromatin state is defined by combinations of marks. An active promoter shows H three K four me three plus H three K twenty-seven ac. An active enhancer shows H three K four me one plus H three K twenty-seven ac. A poised enhancer has H three K four me one but no H three K twenty-seven ac — ready but not yet active. Polycomb-repressed regions carry H three K twenty-seven me three. Constitutive heterochromatin shows H three K nine me three. By measuring all these marks across the genome, you can annotate every region with its functional state. That is the goal of epigenomic mapping.
§ 4

ChIP-seq

How do we actually map where these marks are in the genome? ChIP-seq.

ChIP-seq protocol

  1. Crosslink proteins to DNA with formaldehyde
  2. Fragment chromatin (~200–500 bp)
  3. Add antibody · pull down protein-DNA complexes
  4. Reverse crosslinks · purify DNA · sequence
  5. Map reads · call peaks
ChIP-seq stands for chromatin immunoprecipitation followed by sequencing. The protocol has five steps. One. Crosslink proteins to D N A in living cells using formaldehyde. This freezes protein-D N A interactions in place. Two. Lyse the cells and fragment the chromatin into pieces of two hundred to five hundred base pairs. Three. Add an antibody specific to your target protein — a histone mark, a transcription factor — and use magnetic beads to pull down the protein-D N A complexes. Four. Reverse the crosslinks, purify the D N A, and sequence it. Five. Map the reads to the genome and call peaks — regions with significantly higher read density than expected by chance.

The result · peak profiles

The ChIP-seq workflow shown here. Crosslinking captures protein-D N A interactions. Sonication fragments the D N A. The antibody specifically pulls down complexes containing your target. Sequencing reveals the genomic locations where the protein was bound. The output is a peak profile — sharp peaks at binding sites, low signal elsewhere. ChIP-seq for H three K twenty-seven ac maps active enhancers and promoters. ChIP-seq for C T C F maps insulator sites and loop anchors. ChIP-seq for a transcription factor like p fifty-three maps every place that factor regulates genes.
§ 5

ATAC-seq

Now a different approach. Instead of immunoprecipitating one protein at a time, ATAC-seq measures all the open chromatin in the genome at once.

ATAC-seq · the Tn5 trick

  • Tn5 transposase loaded with sequencing adapters
  • Cuts DNA only in accessible chromatin
  • Inserts adapters in the same step (tagmentation)
  • Result: open regions sequenced · closed regions invisible
ATAC-seq stands for Assay for Transposase-Accessible Chromatin. It exploits a hyperactive bacterial transposase called Tn five. The trick is that Tn five can only access D N A that is not wrapped tightly around nucleosomes. Where chromatin is open, Tn five binds and cuts. Where chromatin is closed, Tn five cannot reach. In a single reaction called tagmentation, Tn five simultaneously cuts the D N A and inserts sequencing adapters. The fragments that result are exactly the regions of the genome that were accessible. Sequence them, and you have a genome-wide map of open chromatin.

ATAC-seq output

Here is the ATAC-seq workflow. Tn five enters the permeabilised nucleus, cuts and tags accessible D N A, and the fragments are amplified and sequenced. The output is sharp peaks at open promoters, enhancers, and other regulatory elements. ATAC-seq is fast — about three hours from cells to library — and works with very few cells, as little as five hundred. It also requires no antibody. By looking at transcription factor motifs enriched within ATAC peaks, you can infer which factors are active in a cell type, even without ChIP-seq data for them. ATAC-seq is now the standard first-pass tool for mapping the regulatory landscape of any new cell type.
§ 6

CUT&RUN

ATAC-seq is great for accessibility. ChIP-seq is great for specific proteins. CUT and RUN combines the best of both.

CUT&RUN · targeted cleavage

  • No crosslinking · no sonication
  • Antibody binds target · pA-MNase fusion binds antibody
  • Calcium activates MNase → cuts only at the binding site
  • Tiny fragments diffuse out · sequenced
CUT and RUN is a newer method that improves on ChIP-seq. The protocol does not use crosslinking or sonication, both of which can introduce artifacts. Instead, intact cells are gently permeabilised. An antibody binds the target protein. A protein A-micrococcal nuclease fusion, called pA-MNase, then binds the antibody. When calcium is added, the nuclease activates and cuts the D N A — but only right at the binding site, since that is where the antibody and pA-MNase are positioned. Small D N A fragments diffuse out of the nucleus into the buffer, where they are collected and sequenced.

Why CUT&RUN beats ChIP-seq

Compared to ChIP-seq, CUT and RUN gives sharper peaks because the cutting is precisely targeted. It produces lower background because only fragments near the antibody are released. It works with far fewer cells — as few as one hundred, compared to millions for ChIP-seq. And it avoids crosslinking artifacts. For these reasons, CUT and RUN has become the preferred method for many transcription factors and for studies of rare cell populations. Standard ChIP-seq is still used heavily, especially for histone marks, but CUT and RUN is gaining ground rapidly.

Integrated Epigenomics

Now let's put all the layers together. Each method gives a piece of the puzzle. Combined, they give the complete picture.

Each method · one layer

MethodWhat it measures
RNA-seqGene expression
scRNA-seqSingle-cell expression
ChIP-seqProtein-DNA binding
ATAC-seqChromatin accessibility
CUT&RUNHigh-res protein binding
Hi-C3D contacts
Each method illuminates one regulatory layer. R N A-seq tells you which genes are expressed. Single-cell R N A-seq adds heterogeneity. ChIP-seq maps where transcription factors and histones bind. ATAC-seq maps open chromatin. CUT and RUN gives high-resolution binding maps. Hi-C, which we did not cover today, measures three-dimensional chromatin contacts. Each is necessary. None is sufficient. Together, they give the complete picture.

An active enhancer · multi-track signature

  • ATAC-seq: open
  • H3K27ac ChIP-seq: active mark
  • H3K4me1 ChIP-seq: enhancer mark
  • RNA-seq nearby gene: highly expressed
Here is what an active enhancer looks like when you align all the data tracks. The ATAC-seq track shows an open peak — accessible chromatin. The H three K twenty-seven ac ChIP-seq track shows the active mark. The H three K four me one ChIP-seq track confirms enhancer identity. R N A polymerase two is not at the enhancer itself but is loading at the nearby promoter. R N A-seq shows the target gene is highly expressed. The combined signature — open + H three K twenty-seven ac + H three K four me one + active target — is unambiguous evidence of an active enhancer.

A repressed gene · the opposite signature

  • ATAC-seq: closed
  • H3K27me3 ChIP-seq: repressive mark
  • H3K4me3: absent
  • RNA-seq: no expression
The opposite signature defines a Polycomb-repressed gene. ATAC-seq shows closed, inaccessible chromatin. H three K twenty-seven me three ChIP-seq is enriched. H three K four me three, the active promoter mark, is absent. R N A-seq shows the gene is silent. This signature is typical of developmental genes that are turned off in the wrong cell type — for example, liver-specific genes in a neuron. The Polycomb system keeps them silenced but ready to be reactivated if conditions change.
§ 8

Why It Matters
for Genetics

Let's connect this back to medical genetics.

Most disease variants are regulatory

  • ~93% of GWAS hits are non-coding
  • Often disrupt enhancer activity
  • Small expression changes → real disease risk
Recall from earlier in Part Five — about ninety-three percent of G W A S hits sit in non-coding regions, mostly in enhancers. They typically disrupt transcription factor binding sites, weakening enhancer activity, and reducing target gene expression. The change in expression may be only ten or twenty percent, but multiplied across many such variants, the effect on disease risk becomes substantial. To interpret these variants, you need the regulatory annotation. ATAC-seq, ChIP-seq, and CUT and RUN data tell you which non-coding regions are functional enhancers, in which cell types, and what gene they control.

Type 2 diabetes example

  • Most T2D risk variants in pancreatic islet enhancers
  • Reduce insulin gene expression by 10–20%
  • Each variant: tiny effect
  • ~600 variants combined → significant disease risk
Concrete example. Most type two diabetes risk variants from G W A S sit in enhancers active in pancreatic islet cells — beta cells specifically. Each variant slightly reduces enhancer activity and therefore slightly reduces insulin gene expression. Each individual variant has a tiny effect. But over six hundred such variants combined produce a substantial increase in disease risk. The diagnostic and therapeutic implications follow directly from the regulatory architecture. You need ATAC-seq and ChIP-seq data from beta cells to make sense of any of this.

Therapeutic targets

  • Insufficient expression → CRISPRa (Ch. 27)
  • Excess expression → CRISPRi
  • Wrong epigenetic state → epigenetic editors
  • Therapy follows the regulatory architecture
Once you understand the regulatory architecture of a disease, therapy design follows naturally. If a disease is caused by insufficient expression, as in S C N two A haploinsufficiency, use CRISPR a to boost expression of the normal allele. If excess expression — too much M Y C in cancer — use CRISPR i. If the chromatin state is wrong, use epigenetic editors to write or erase specific histone marks. The toolkit is now extensive, and it maps directly onto the regulatory layers we have discussed all chapter.

Summary

Let's wrap up.

What to take away

  • RNA-seq — gene expression genome-wide
  • scRNA-seq — cell-type heterogeneity
  • ChIP-seq — protein-DNA interactions
  • ATAC-seq — open chromatin · fast · low input
  • CUT&RUN — high resolution protein binding
  • Together → complete regulatory architecture · disease interpretation · therapy design
Six takeaways. One. R N A-seq measures gene expression genome-wide. Two. Single-cell R N A-seq reveals cellular heterogeneity hidden in bulk experiments. Three. ChIP-seq maps where specific proteins bind. Four. ATAC-seq maps open chromatin without antibodies, fast and with low input. Five. CUT and RUN gives high-resolution protein-D N A binding maps. And six. Combined, these methods reveal the complete regulatory architecture of any cell type — necessary for interpreting disease variants and designing therapies.
